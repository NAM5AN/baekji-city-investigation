# 감각 특성 확장 계획

현재 관찰 공유는 같은 현장에 있는 일반 캐릭터의 기본 시야·청력을 기준으로 한다.

추후 캐릭터 데이터에 `perception.vision`과 `perception.hearing` 등급을 추가하고, 관찰문 생성 요청에 관찰자별 감각 범위를 전달한다. 좋은 청력은 낮은 목소리나 먼 소리를, 좋은 시력은 작은 손동작이나 먼 거리의 행동을 추가로 인지할 수 있게 한다. 기본 캐릭터는 큰 소리의 직접 발화만 내용까지 듣고, 혼잣말·중얼거림의 내용은 알 수 없다.

이번 버전에서는 데이터 필드와 판정 규칙을 구현하지 않는다.
