# v0.3.18 폰트 역할표

| 번호 | 파일 | 적용 역할 |
| --- | --- | --- |
| 1 | `01-Ycomputer.otf` | 본문·SYSTEM·입력창·일반 UI |
| 2 | `02-DenkiChipHangul.woff2` | 페이지·장소·팝업 타이틀 |
| 3 | `03-DOSPilgi.ttf` | 지도 메모 등 짧은 필기 표현 |
| 4 | `04-Suhgung16.woff2` | 오브젝트 조사 결과 |
| 5 | `05-Ramche.ttf` | 메모·주의문 |
| 6 | `06-MonaS12TextKR.woff2` | 조사 기록 보조문 |
| 7 | `07-Cafe24PROUP.woff2` | 짧은 배지·상태 수치·창 라벨 |

SYSTEM 출력 내용과 시간 표시는 항상 1번 Y콤퓨타체 본문 폰트를 사용합니다. 3~6번 필기체는 긴 로그나 일반 UI에 순환 적용하지 않고, 조사 결과·메모·기록처럼 성격이 분명한 짧은 영역에만 사용합니다. 7번 포인트 폰트도 일반 `strong`·버튼·탭 전체에 강제하지 않습니다.

3번 `fonts-master.zip`은 중앙 디렉터리와 뒤쪽 TTF 일부가 손상되어 있었으나, 완전하게 남아 있던 `DOSPilgi-16.bdf` 비트맵을 복구해 브라우저용 TTF로 변환했습니다.

제공 ZIP에 포함된 라이선스 문서는 `licenses/`에 함께 보관했습니다. 1번 Y콤퓨타체는 원본 서체 가이드의 라이선스 정책을 함께 포함했으며, 원본 OTF를 수정·변환하지 않았습니다. 5번 ZIP에는 별도 라이선스 문서가 포함되어 있지 않았습니다.
